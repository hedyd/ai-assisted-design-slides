05 — Local vs. Remote

Two ways to
work with Claude.

Local
  • Your laptop
  • Fast & private
  • Works offline
  • Claude edits files directly in VS Code
  • You see every change as it happens
claude
Remote
  • Anthropic's cloud computers
  • Session persists if you close your laptop
  • Claude clones your repo, does the work, pushes back
  • Great for bigger tasks you can walk away from
claude --remote

Hosting alternatives: Netlify, GitHub Pages

06 — Skills & CLAUDE.md

Two ways to shape
how Claude behaves.

Personal · Manual
CLAUDE.md

A plain text file you create yourself in the project root. Claude reads it automatically at the start of every session.

Put your fonts, colors, image naming rules, and preferences here. Think of it as a sticky note on your desk.

You write it. It's yours.
Community · Installable
Skills

Reusable instruction packs made by others — like plugins for Claude's behavior. Install from skills.sh.

npx skills add portfolio-site
Someone else wrote it. Anyone can use it.
