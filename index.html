02 — System Prompt

Your starting prompt.

Paste into Claude Chat and upload your resume at the same time.

I want to build a professional portfolio website using Tailwind CSS and deploy it on GitHub Pages.

Please help me create a complete, single-page portfolio with these sections:
1. Hero — photo, name, title, and introduction
2. Skills — list and badges
3. Projects — summary, responsibilities, key features, and skills learned
4. Contact — social media links

Requirements
- Clean, modern, responsive design
- Monochromatic colour palette (suggest options)
- Single HTML file with embedded CSS
- Use https://lucide.dev/icons/ for iconography
- Optimised for GitHub Pages

I'll upload
- Resume (PDF or DOCX)
- Professional photo
- Project images
- LinkedIn profile link

Please
1. Extract information from my resume and LinkedIn page
2. Build the website structure
3. Ask me questions to customise the design
4. Provide a complete ZIP file ready for deployment
5. Include GitHub Pages deployment instructions

Let's start! I'm uploading my resume now.

05 — Local vs. Remote

Two ways Claude
Code works.

LOCAL
  • Claude runs on your laptop, editing files directly inside VS Code
  • Fast and immediate — you watch it happen in real time
  • Your machine has to stay on
REMOTE
  • Claude runs on Anthropic's cloud computers, not yours
  • Give it a task, close your laptop, come back later
  • Clones your GitHub repo, works in a secure cloud environment, pushes changes back automatically
  • Monitor sessions from the Claude mobile app

Tip: Switch between local and remote directly in the Claude Code chat — no terminal needed

08 — Vercel

Publishing
your site.

A platform that takes your GitHub repo and turns it into a live website — automatically, on every push.

1 Connect Vercel to your GitHub repo
2 Push any change — or Claude's remote session pushes for you
3 Your site auto-updates — get a live URL in ~60 seconds
Free for personal projects

Alternatives: Netlify · GitHub Pages
