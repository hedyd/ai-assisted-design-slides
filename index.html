05 — Local vs. Remote

Two ways Claude
Code works.

LOCAL
  • Claude runs on your laptop, editing files directly inside VS Code
  • Fast and immediate — you watch it happen in real time
  • Your machine has to stay on
REMOTE
  • Claude runs on Anthropic's cloud computers, not yours
  • Give it a task, close your laptop, come back later
  • Clones your GitHub repo, works in a secure cloud environment, pushes changes back automatically
  • Monitor sessions from the Claude mobile app

Tip: Switch between local and remote directly in the Claude Code chat — no terminal needed

07 — Skills

Giving Claude
instructions.

Skills are installable instruction packs built by the community — they teach Claude how to do specific things well.

What skills can do
Examples

Build a portfolio layout · follow a design system · generate accessible HTML

Browse and install from skills.sh — Claude picks them up automatically.

Made by the community. Anyone can use them.
How to add a skill
Install
npx skills add https://github.com/anthropics/skills --skill frontend-design

Or browse skills.sh — install with one command and Claude picks them up automatically.

08 — Vercel

Publishing
your site.

A platform that takes your GitHub repo and turns it into a live website — automatically, on every push.

1 Connect Vercel to your GitHub repo
2 Push any change — or Claude's remote session pushes for you
3 Your site auto-updates — get a live URL in ~60 seconds
Free for personal projects

Alternatives: Netlify · GitHub Pages

Demo: Connect Vercel to your GitHub repo → get a live URL in ~60 seconds
